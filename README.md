    Clube da Leitura Digital
A proposta de desenvolviento de conehcimentos e habilidas na prototipação do Front-End, tem por objetiov atender as necessidades de um site de leitura e              comercialização de livros pelo mundo, a meta de conclusão é aplicar todas as funcinalidades coerentes para aplicar o uso do site e garantir que sua estrutura         nativa esteja em perfeita harmonia com a parte visual da platafroma.

    Tecnologias Instaladas
HTML (Semântica e Acessibilidade)
CSS (Estilização e Feedback Visual)
JavaScript (Manipulação de DOM e Eventos)

    Estrutura do Projeto
O projeto foi organizado da segunite maneira: 
├── index.html 
└── src
 ├── styles/ (Estilos da página)   
 ├── scripts/ (Lógica de interação) 
 └── images/ (Ativos visuais)

    Funcionalidades Implementadas
Interatividade Visual: Elementos que reagem ao passar do mouse respondendo as solicitações de alteraçã.
Saudação Dinâmica: Mensagem de boas-vindas que muda conforme o horário local do usuário, apresentandoa frase correpondente com a configuração do horário.
Cálculo em Tempo Real: Formulário que processa e exibe resultados assim que o usuário digita, apresntando listas e tabelas solicitadas.
 Injeção de Conteúdo: Adição de novos itens na tela utilizando manipulação de DOM.
